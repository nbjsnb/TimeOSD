# TimeOSD 功能说明（V1.1）

## 项目定位

面向运动相机无时间戳素材的后处理，实现“补时 + OSD 导出”工作流。

## 核心能力

1. 支持缩时模式与正常速率模式。
2. 支持 ASS 生成、加载预览、按频率或按帧更新。
3. 支持拖拽定位、进度条预览、参数实时反馈。
4. 支持 20 秒测试导出与全片导出。
5. 支持配置预设保存/加载/删除。

## V1.1 发布点

- 新增输出模式：硬字幕（重编码）与软字幕 MKV（不重编码）。
- 硬字幕新增“沿用原视频编码类型”。
- 完善 FFmpeg/FFprobe 自动查找与提示。
- 参数区“配置预设”区块移到底部，与导出参数分离。
