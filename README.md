# TimeOSD

## 中文

### 项目简介
TimeOSD 是一个给视频叠加时间 OSD 的桌面工具，适用于延时摄影和运动相机素材后处理。

### v1.2 主要更新
- 中英文双语界面切换（标题栏）
- 启动脚本增加 Python 环境检测
- 缺少 Python 时给出下载链接
- 缺少依赖时提示是否自动安装

### 运行要求
- Windows 10/11
- Python 3.10+（源码运行）
- FFmpeg / FFprobe（使用不含依赖版 EXE 时需要）

### 源码运行
```powershell
python -m pip install -r requirements.txt
python app.py
```

或：
```bat
run_timeosd.bat
```
